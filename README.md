##MAXIMUM CIPHER OVERDRIVE
A game about burning calories made in one hour at the 2016 Fall UC Davis game jam!

#Team
- Tanveer Bariana
- Justin Mendiguarin
- Nik Sorvari

#Description
Survive! Use the left and right arrowkeys to move around. Press space to jump.

#Controls
Left, Right, Space

#Assets
